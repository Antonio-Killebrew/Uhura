# Final Project Proposal

## https://github.com/Antonio-Killebrew/project

## Description
This program will generate an interactive video game for the user to play.
The game will be a 2D platforming game similar to Mario Bros.

## Features
- Jumping
    - The player will jump using the up arrow key.
    - This will be executed using a function called move,
        using the player y position and custom physics functions.
- Platforms
    - The player will be able to traverse platforms.
    - Platforms will be executed using collision functions.
- Enemies
    - Enemies will use the same collision functions as the platforms
    - The player will jump over and avoid the enemies in the game.

## Challenges
- I will need to recall and learn more about the draw functions and tile maps.
- Adding traps, player attacks, and items would require some extra time.
- Proper syntax and organization will be important for this many lines of code.

## Outcomes
Ideal Outcome:
- The game will be fully functional for the player to use for any amount of time.

Minimal Viable Outcome:
- At the very least the game will feature a player character that can move across platforms.

- Week 2
    1. Implement collision and jumping mechanics.

- Week N (Final)
